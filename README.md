### ğŸ§  Alur Pengerjaan (Workflow Git)
1. **Inisialisasi Git:**  
   Membuat repository lokal menggunakan perintah `git init`.  

2. **Commit Bertahap:**  
   Melakukan commit untuk setiap bagian halaman (Home, Projects, Contact, Comment, dan Footer).  

3. **Pembuatan Branch Baru:**  
   Membuat branch `feature-styling` untuk eksperimen tampilan website.  

4. **Merge Branch:**  
   Menggabungkan hasil eksperimen dari `feature-styling` ke branch utama `master`.  

5. **Push ke GitHub:**  
   Mengunggah repository lokal ke GitHub menggunakan perintah `git push`.
